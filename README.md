# TimeM

##工具介绍##
人人影视的时间机器。制作字幕使用的万能工具


##说明##
此仓库内的源代码可经由Visual Studio 2010编译。删除了导出到Mac的功能

##下载##
Release下有我编译的文件。如果你不相信我，也可以自行编译。

##使用条件##
安装vodsub即可

## 更新日志 ##
修改logo为新版、修正链接等

##Bug##
打开视频选项无法同时打开字幕，需要再次打开字幕才行。原版没有这个问题。
